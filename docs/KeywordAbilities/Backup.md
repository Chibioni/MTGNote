# Backup(N) / 賛助(N)

* このクリーチャーが戦場に出たとき、クリーチャー1体を対象とする。それの上に [[One One Up Counter]] をN個置く。それが他のクリーチャーである場合、それはターン終了時まで、この能力より下に印刷されている賛助でないすべての能力を得る。

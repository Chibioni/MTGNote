# Unearth(N) / 蘇生(N)

(N) : あなたの墓地にあるこのカードを戦場に戻す。これは [[Haste]] を得る。
次の終了ステップの開始時に、あるいは、これが戦場を離れるなら、これを追放する。
蘇生はソーサリーとしてのみ行う。

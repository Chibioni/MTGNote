# Bitter Chill / 厳寒 (1)(青)

* #エンチャント
  * #クリーチャー

* 厳寒が戦場に出たとき、エンチャントしているクリーチャーをタップする。
* エンチャントしているクリーチャーは、それのコントローラーのアンタップ・ステップにアンタップしない。
* 厳寒が戦場から墓地に置かれたとき、(1)を支払ってもよい。そうしたなら、 [[Scry]] 1を行い、その後、カード1枚を引く。

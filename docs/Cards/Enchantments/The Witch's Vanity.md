# The Witch's Vanity / 魔女の虚栄 (1)(黒)

* #エンチャント
  * #英雄譚

* (この英雄譚が出た際とあなたのドロー・ステップの後に、伝承カウンター1個を加える。IIIの後に、生け贄に捧げる。)
